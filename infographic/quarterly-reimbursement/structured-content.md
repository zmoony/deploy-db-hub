# Structured Content: 季度报销去哪儿了？

## Title
季度报销去哪儿了？

## Learning Objectives
用幽默方式表达员工对公司拖延季度报销的无奈和期待

## Sections

### Section 1: 承诺很美
- Key concept: 公司承诺季度报销
- Content: "放心，季度报销，说到做到！"
- Visual element: 老板/HR微笑着画大饼的卡通形象，手里拿着"季度报销"的承诺牌子
- Text labels: "公司承诺"、"季度报销"

### Section 2: 现实很骨感
- Key concept: 季度过去了，报销还没影
- Content: "季度都过完了，我的报销呢？"
- Visual element: 日历翻页，春夏秋冬都过了，员工坐在空荡荡的办公桌前，钱包空空
- Text labels: "Q1过去了"、"Q2过去了"、"报销还在路上..."

### Section 3: 员工的内心戏
- Key concept: 焦急等待的各种阶段
- Content: 
  - "第1天：没关系，流程需要时间"
  - "第30天：应该快了吧？"
  - "第90天：我是不是忘了什么？"
  - "第180天：报销是什么？我不记得了"
- Visual element: 员工表情包式的情绪变化：期待 → 疑惑 → 焦虑 → 佛系
- Text labels: 情绪时间轴

### Section 4: 灵魂拷问
- Key concept: 直接但幽默地表达诉求
- Content: "老板，我的报销，它还在吗？"
- Visual element: 员工拿着放大镜找"报销款"，或者举着"寻物启事：我的报销"
- Text labels: "寻报销启事"、"重金酬谢"（其实就是要回自己的钱）

## Data Points
- 季度 = 3个月
- 员工等待时间远超预期
- 报销款 = 员工的钱

## Design Instructions
- 使用 kawaii 或 claymation 卡通风格
- 色彩鲜明但温馨
- 加入对话气泡
- 使用表情符号式的角色表情
- 整体轻松幽默，不尖锐批评
